# Лента объявлений

Веб-сервис для просмотра объявлений с бесконечной подгрузкой, регистрацией и аутентификацией.

---

## Стек технологий

| Технология | Версия |
|------------|--------|
| Java | 21 |
| Spring Boot | 3.2.0 |
| Spring Security | 6.2.0 |
| Spring Data JPA | 3.2.0 |
| PostgreSQL | 15+ |
| Thymeleaf | 3.1.2 |
| Maven | 3.9+ |

---

## Быстрый запуск (через готовый JAR)

1. Скачайте файл `adboard-1.0-SNAPSHOT.jar` из раздела [Releases](https://github.com/ksusha334/adboard/releases)
2. Убедитесь, что у вас установлена Java 21
3. Запустите из командной строки:

```bash
java -jar adboard-1.0-SNAPSHOT.jar
```

4. Откройте браузер: `http://localhost:8080`

---

## Запуск из исходников (сборка через Maven)

### Требования

- Java 21 или выше
- PostgreSQL 15 или выше
- Maven 3.9 или выше

### 1. Создать базу данных

```bash
createdb -U march adboard
```

(замените `march` на ваше имя пользователя PostgreSQL)

### 2. Настроить подключение к БД

Файл `src/main/resources/application.properties`:

```properties
spring.datasource.url=jdbc:postgresql://localhost:5432/adboard
spring.datasource.username=march
spring.datasource.password=
spring.jpa.hibernate.ddl-auto=update
server.port=8080
```

### 3. Собрать проект

```bash
mvn clean package
```

### 4. Запустить приложение

```bash
java -jar target/adboard-1.0-SNAPSHOT.jar
```

### 5. Открыть в браузере

```
http://localhost:8080
```

### 6. Зарегистрироваться и войти

- Перейдите на `http://localhost:8080/register`
- Создайте учётную запись
- Войдите на `http://localhost:8080/login`

При первом запуске автоматически создадутся 100 тестовых объявлений.

## Тест-кейсы

| № | Сценарий | Ожидаемый результат |
|---|----------|---------------------|
| 1 | Регистрация нового пользователя | Редирект на /login с сообщением об успехе |
| 2 | Вход с корректными данными | Редирект на главную страницу с лентой |
| 3 | Вход с некорректными данными | Ошибка "Неверное имя пользователя или пароль" |
| 4 | Скролл вниз на ленте | Подгрузка новых объявлений |
| 5 | Сортировка по цене (дешёвые) | Объявления от меньшей цены к большей |
| 6 | Поиск по названию | Отображаются только объявления, содержащие текст |
| 7 | Клик на карточку объявления | Открывается страница с полной информацией |
| 8 | Смена сортировки | Лента перезагружается в новом порядке |
| 9 | Выход из системы | Редирект на /login |

## Функционал

### Обязательный (по ТЗ)

- Регистрация и аутентификация пользователей
- Просмотр списка объявлений с пагинацией (порциями)
- Бесконечная подгрузка при скролле
- Сортировка по дате (новые сверху)
- Просмотр отдельного объявления с полной информацией
- Отображение статусов загрузки (спиннер, ошибка, конец списка)
- Обработка ошибок сети с кнопкой "Повторить"

### Дополнительный (для повышенной оценки)

- Кэширование загруженных страниц (ConcurrentHashMap)
- Сортировка по цене (дешёвые / дорогие)
- Сохранение позиции скролла в localStorage

```bash
export JAVA_HOME=/Library/Java/JavaVirtualMachines/jdk-21.jdk/Contents/Home

```
